# NotebookLite Welcome Guide

Welcome to the **NotebookLite** document repository! This file is hosted directly in your GitHub repository and pre-loaded automatically into your AI chat context.

## How it works on GitHub
When this page is hosted on **GitHub Pages**, the client-side code fetches this file directly from your repository path using standard HTTP requests. This means:
1. You can add, edit, or delete markdown/text files in the repository.
2. Update the file list array at the top of `app.js`.
3. When users load the page, the app automatically fetches the latest contents of all listed repository files and populates their workspace sources checklist.

## Quick Tips
- You can toggle this document off in the left sidebar to exclude it from the AI context.
- Hover over inline citations generated by the model to see snippets of this document.
